# GunDamagePlus / 枪械增强插件

[Englsh](README.md)

---

**GunDamagePlus** 是一个 _LeviLamina_ 插件，旨在修复 Minecraft 基岩版原版中自定义 Addon 枪械的常见问题。移除抛掷物（子弹）伤害的无敌帧，从而实现真正的连射伤害，并且支持记录伤害来源

### 功能

1.  **击杀提示修复**：本插件会在抛掷物初始化时强制将主人（Owner）设置为开枪的玩家。击杀信息将正确显示玩家名字，并能触发相关的脚本事件

2.  **移除受击无敌帧**：本插件专门针对抛掷物类型的伤害移除了无敌冷却。每一发命中的子弹都会造成伤害，让机枪类武器不再刮痧

### 安装

1.  下载压缩包
2.  将压缩包解压到服务器目录下的 `plugins` 文件夹中
3.  启动服务器即可
